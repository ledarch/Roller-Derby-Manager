Comment fonctionne Roller derby manager?

Configurez

Créez votre compte sur Roller derby manager et renseignez les différentes informations requises avant de pouvoir débuter l'expérience.

Rejoignez

Recherchez votre équipe et faîtes la demande pour l'intégrer ou saississez le code fournit par le responsable d'équipe.

Naviguez

Accéder aux calendriers de l'équipe, renseigner votre présence sur les différents évènements, visualiser les fichiers statistiques, le playbook...

Fichiers

Retrouvez tous les fichiers mis à disposition par votre club, fichiers statistiques, playbook...

Vous souhaitez créer votre club?

Configurez

Créez votre compte sur Roller derby manager et précisez être responsable du club puis saississez les informations nécessaire.

Invitez

Ne tardez plus et envoyez le code permettant à vos joueurs de rejoindre l'équipe !

Gestion

Dès à présent vous pouvez gérer vos effectifs, créer des événements et programmer les rappels de présence, stockez des fichiers.

Stockez

Retrouvez facilement vos fichiers ! Archivage des stats de chaque match, des guides de stratégie ou tout autres documents.
